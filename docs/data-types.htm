Data Types

Enum

JavaScript does not have an enum type, so we have to use a work-around.

Using an Array

We will use an array with both numerical and string keys:

const Color = ['red','green','blue'];
Color.red = 0;
Color.green = 1;
Color.blue = 2;
console.log( Color );
> [ 'red', 'green', 'blue', red: 0, green: 1, blue: 2 ]

This array can be used to translate color names into numerical values, and vice versa. The code above can be streamlined with an immediately invoked function expression (IIFE):

(function (Color) {
	Color[Color["red"]=0]='red';
	Color[Color["greeb"]=0]='green';
	Color[Color["blue"]=0]='blue';
	})(Color || (Color={}));
console.log( Color );
> [ 'red', 'green', 'blue', red: 0, green: 1, blue: 2 ]

Using an Object

Alternatively, we could use object literal notation to create an object, like so:

const Color = {
	0: 'red' , red: 0,
	1: 'green' , green: 1,
	2: 'blue' , blue: 2
	} ;
console.log( Color );
> color
> { '0': 'red', '1': 'green', '2': 'blue', red: 0, green: 1, blue: 2 }
> color[0]
> 'red'
> color['0']
> 'red'
> color.red
> 0
> color['red']
> 0
